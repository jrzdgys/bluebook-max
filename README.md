# 蓝宝书Max

> 基于 Alpha派 蓝宝书的机构级投研情报工具。数据驱动架构，自动化分类与评分。

## 数据架构（v2）

```
模板 (template) ← 固定 CSS/JS，不随版本变化
  ↓ 加载
数据JSON ← 每版独立，包含主题/标的/评分/收盘价
  ↓ 渲染
HTML页面 ← 纯客户端渲染，无构建步骤
```

新增版本只需替换 JSON 数据块，模板不动。

## 标的分类规则（决策树）

分类完全由 **Alpha派原文的组序和角色定位** 驱动，不依赖人工判断。

| 规则 | 条件 | 分类 |
|------|------|------|
| R0 | 市值 < 50亿 | 不能为龙头（可信度底线） |
| R1 | 原文第一组首位 + 市值 ≥ 50亿 | 🔴 龙头首选 |
| R2 | 原文第一组（非首位） | 🟠 弹性机会 |
| R3 | 原文第二组 + 角色含"核心/直接/龙头/充分" | 🟠 弹性机会 |
| R4 | 市值 > 300亿 + 主题内排名前40% | 🟠 弹性机会 |
| R5 | 其余所有 | 🔵 相关标的 |

**核心原则：** Alpha派原文是机构共识的映射，股票在原文中的出现顺序和分组本身就是市场对其重要性的排序。规则只做两件事：① 读懂这个排序；② 用市值做底线过滤。

## 主题评分规则（三层独立评分）

每个主题的热度总分由三个**各自有独立数据源**的维度加总，不做机械拆分。

```
热度总分(0-100) = 机构关注度(0-60) + 市场信号(0-25) + 催化强度(0-15)
```

### 第一层：机构关注度（0-60分）

| 数据源 | 计算 |
|--------|------|
| Alpha派 index 值 | `index / 10 × 60` |

Alpha派的 index 是机构编辑对话题综合重要性的判断。index=10 → 60分，index=5 → 30分。

### 第二层：市场信号（0-25分）

| 数据源 | 计算 |
|--------|------|
| 腾讯行情 API | 该主题全部标的的**平均涨跌幅绝对值** |

| 均价涨幅 | 得分 |
|----------|------|
| ≥5% | 25 |
| ≥3% | 18 |
| ≥1% | 10 |
| <1% 或无数据 | 5 |

反映市场是否已经在交易这个主题。

### 第三层：催化强度（0-15分）

| 数据源 | 计算 |
|--------|------|
| Alpha派 summary 文本 | 关键词匹配分析 |

| 催化剂类型 | 得分 | 关键词示例 |
|------------|------|-----------|
| 确定性催化剂 | 15 | 官宣、确认、正式发布、业绩、获批、正式 |
| 高置信度预期 | 10 | 或将、预计、接近、计划、量产、推进 |
| 传闻/跟踪 | 8 | 传、据称、关注、跟踪、预期 |

### 阶段判定

| 总分 | 阶段 |
|------|------|
| ≥85 | 🔥 主升 | 机构高度聚焦，催化剂已触发 |
| ≥65 | 📈 强化 | 新催化加入，关注度显著上升 |
| ≥50 | ➡️ 持续 | 维持关注，平稳运行 |
| <50 | 🌱 孵化 | 早期信号，待催化确认 |

## 摘要与引用精简规则

从Alpha派原始摘要中**全自动提取**，不依赖历史版本缓存。每版生成时实时执行。

### 摘要提取（5条规则）

| 规则 | 操作 | 说明 |
|------|------|------|
| R1 | 去除时间/背景引导语 | `"6月22日，行业专家..."` → `"行业专家..."`；删除"在过去24小时内"、"本周"、"昨夜今晨（...）"等 |
| R2 | 去除元论述过渡句 | 识别并剥离"这一事件的边际变化在于"、"这标志着"、"核心边际变化在于"等引导词，保留后续实质内容 |
| R3 | 保留核心事件（≤85字） | 取前1-2句事实性陈述，控制长度 |
| R4 | 去除Markdown标记 | 移除所有 `**...**` 加粗标记 |
| R5 | 移除"关注："段 | 摘要提取时排除"关注：**股票A**/**股票B**（理由）"的标的列举段 |

### 引用提取（4条规则）

| 规则 | 操作 | 说明 |
|------|------|------|
| Q1 | 去除元论述引导语 | 同R2逻辑 |
| Q2 | 去除Markdown标记 | 同R4 |
| Q3 | 优先高冲击力论点 | 评分权重：对比结构（不是…而是…）> 动词驱动（正在兑现）> 关键词（核心/关键/稀缺） |
| Q4 | 控制在60字以内 | 超长则在自然断点截断 |

## 股票推荐理由提取规则

**照抄Alpha派原文，不做任何修改。**

### 提取逻辑

Alpha派原文的"关注："段格式为：
```
**股票A**/**股票B**（推荐理由），**股票C**（推荐理由2）
```

提取规则：
1. 定位"关注："后的内容段
2. 识别 `**股票名**` 标记提取股票名称
3. 识别 `（推荐理由）` 中文括号内容作为该组股票的理由
4. `/` 分隔的同组股票共享同一理由
5. 每个topic独立提取，同一股票在不同topic可有不同理由

### 原则

- **原文照搬**：不缩写、不改写、不重新措辞
- **不编造**：如原文未提供理由，使用所属分类标签（龙头首选/弹性机会/相关标的）
